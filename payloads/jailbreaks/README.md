# Jailbreaks — named payloads

Verbatim openers for published jailbreak families. See
[../../3-exploitation/common/jailbreaks.md](../../3-exploitation/common/jailbreaks.md)
for the discussion of when each works.

## Files

- [dan.md](dan.md) — DAN family (12+ revisions, just the openers)
- [aim.md](aim.md) — AIM (Always Intelligent / Machiavellian)
- [developer-mode.md](developer-mode.md) — "developer / debug mode" openers
- [fiction-frames.md](fiction-frames.md) — story / screenplay / character JB
- [multi-turn-ramps.md](multi-turn-ramps.md) — multi-turn opener sequences
- [encoding-bypass.md](encoding-bypass.md) — encoded JB requests

> ⚠️ These payloads are widely-published, well-trained-against on modern
> models. I expect modest success on smaller/older/fine-tuned models and
> low success on flagship 2026 models without combination.
