# Templates

Copyable starting points. I drop these into other folders and fill in,
rather than starting a blank file under exam pressure.

| Template | Copy to | When |
|----------|---------|------|
| [scenario-note.md](scenario-note.md) | `scenarios/<YYYY-MM-DD>-<slug>.md` | Start of each new lab / target |
| [finding-writeup.md](finding-writeup.md) | Inline into the final report (or staged under `5-reporting/<finding>.md`) | Each confirmed finding |

These are seeded reference material — I edit them only when I find a gap
during real use, then carry the improvement forward. Day-of-exam I copy,
I don't edit.
